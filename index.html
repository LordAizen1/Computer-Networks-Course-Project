Multi-threaded TCP Chat Server

A production-ready, multi-threaded C++ TCP chat server with advanced features.

View on GitHub

Features

Core Functionality

  • Multi-threaded architecture for concurrent clients
  • Public and private messaging with @user
  • Active user listing via /list
  • Graceful disconnect handling
  • Comprehensive server event logging

File Transfer Protocol

Robust client-server-client streaming implementation.

  • Cross-machine file transfer support
  • Filename and extension preservation
  • User-specific directory organization
  • Timestamp-based conflict prevention
  • Real-time progress tracking

Message Encryption

Symmetric XOR cipher implementation for message confidentiality.

  • Lightweight encryption overhead
  • Protection against packet sniffing
  • Toggle-able encryption mode
  • Security-conscious design
