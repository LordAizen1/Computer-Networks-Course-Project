A production-ready, multi-threaded C++ TCP chat server with advanced features.

Core Features
  • Multi-threaded server for concurrent clients.
  • Public and private messaging (@user).
  • List active users (/list).
  • Graceful disconnect (/quit).
  • Comprehensive server event logging.
Advanced File Transfer

Fixed a critical bug in the original design with a robust client-server-client streaming protocol.

  • Works across different machines.
  • Preserves original filename and extension.
  • Files organized into user-specific directories.
  • Timestamped to prevent filename conflicts.
  • Real-time progress updates.
BONUS: Message Encryption

Implements a symmetric XOR cipher for message confidentiality, demonstrating security awareness.

  • Fast and lightweight encryption.
  • Prevents casual network packet sniffing.
  • Easy to toggle on or off.
  • Shows understanding of core security concepts.
How to Compile & Run

Requires a C++17 compiler (like g++) and make. Works on Linux/macOS.

$ make # In Terminal 1: $ ./server # In Terminal 2: $ ./client
